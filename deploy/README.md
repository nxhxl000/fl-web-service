# deploy

Всё, что связано с запуском системы: `docker-compose.dev.yml` для локальной
разработки (Postgres, Redis, MinIO), `docker-compose.prod.yml` для боевого
стенда (добавляет бэкенд, фронт, inference worker, Caddy для TLS),
шаблоны переменных окружения.

Файлы появятся постепенно по мере прохождения дорожной карты.
