# backend

FastAPI-приложение: REST API, авторизация, проекты, runs, клиенты, метрики,
реестр моделей, оркестрация процесса Flower SuperLink, трансляция live-метрик
по WebSocket.

Внутренние пакеты по доменам: `auth`, `projects`, `clients`, `runs`,
`metrics`, `registry`, `inference`, `orchestrator`.

Запуск в dev-режиме и структура пакетов появятся в следующем коммите.
