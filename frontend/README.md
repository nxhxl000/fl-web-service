# frontend

React-приложение: лендинг, авторизация, дашборд участника, страницы проектов
с live-графиками раундов, реестр обученных моделей, inference playground.

Стек: Vite + React + TypeScript + TailwindCSS + shadcn/ui + TanStack Query,
recharts для графиков, нативный WebSocket API для live-метрик.

Инициализация проекта и базовые страницы появятся в отдельном коммите.
